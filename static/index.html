3x-ui 面板配置

管控页面地址示例: https://IP:6789/panel 或 https://域名:6789/panel

一键部署：自动开放防火墙 80/443/面板端口，安装 SSL 证书，添加默认 VLESS 入站(443)

添加入站 / 入站列表

对已部署的面板，通过 SSH 连接后添加入站或查看入站列表。默认 VMess + gRPC + TLS(443)，证书域名需与 x-ui 18 号菜单安装的域名一致。

新入站配置

若测速报 403：① 证书域名必填且与客户端地址一致 ② 部分测速站会拦截代理 IP，可先访问 google.com 验证

卸载 3x-ui

使用 x-ui 菜单卸载，将移除 3x-ui 面板及 Xray。请确认服务器 SSH 信息正确。
